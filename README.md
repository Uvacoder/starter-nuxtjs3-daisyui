# Nuxt 3 Starter with DaisyUI

Look at the [nuxt 3 documentation](https://v3.nuxtjs.org) to learn more.

## Setup

Install dependency 

```bash
npm i
yarn
```

### Development Server

Start the development server on http://localhost:3000

```bash
yarn dev
npm dev
```

### Production Server

Regular Nuxt with SSR
```bash
yarn build
npm build
```

Prerendered SPA
```bash
yarn generate
npm generate
```
